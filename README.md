# chat-app


##Tech Stack:

###Server side:
1. Express
2. Mongo DB(on mongoLab cloud)
3. Mongoose as model library
4. Redis as Session store
5. Socket.io as message channel
6. Passport with Google outh as authentication

###Client Side
1. React
2. Redux as state store
3. Material-UI
4. Socket.io(of course...)

###Build Tool
1. Webpack

###Deploy
1. Heroku
